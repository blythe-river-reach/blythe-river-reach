Keep this site afloat 🛟

This site is a free, one-person project — the data comes from public sensors, but the domain and upkeep don’t pay for themselves. If the tide table saved you a wasted trip to the ramp, you can toss a few bucks in the tip jar. No account needed, and the site stays free either way.

Build 2026-07-16c — place picker · GitHub data relay · no server functions.
Place timing is estimated from Reclamation river miles and the pulse travel speed (default ~4 mph, editable at the top of this file).
How the data flows: your browser fetches the five USGS gauges live on every load. Reclamation’s servers don’t allow browser access, so a GitHub robot gathers their river sensors (Davis Dam to Martinez Lake), Lake Havasu’s level, and the dam-schedule PDFs about three times an hour and publishes them for this page (the header shows how fresh that data is).
All values are provisional. This is not a flood-warning system. Times are Arizona (MST).
